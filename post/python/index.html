A raspagem de dados é o processo de extrair informações de um site da web. Uma biblioteca popular para realizar a raspagem de dados em Python é o BeautifulSoup.

Introdução 🌟

O BeautifulSoup é uma biblioteca Python que permite analisar e extrair dados de documentos HTML e XML. Ele fornece maneiras convenientes de percorrer a estrutura do documento e encontrar elementos específicos com base em suas tags, atributos e conteúdo.

Instalação ⚙️

Para começar a usar o BeautifulSoup, você precisa instalá-lo em seu ambiente Python. Você pode fazer isso executando o seguinte comando:

pip install beautifulsoup4

Raspagem de Dados ✂️

Para raspar dados de um site com o BeautifulSoup, você precisa seguir algumas etapas:

  1. Importe as bibliotecas necessárias:
    2. import requests
from bs4 import BeautifulSoup
  2. Obtenha o conteúdo HTML da página da web que deseja raspar:
    3. url = 'https://www.example.com'
response = requests.get(url)
html_content = response.content
  3. Crie um objeto BeautifulSoup a partir do conteúdo HTML:
    4. soup = BeautifulSoup(html_content, 'html.parser')
  4. Utilize os métodos do BeautifulSoup para navegar e extrair os dados desejados:
    5. title = soup.title.text.strip()
paragraphs = soup.find_all('p')

Exemplo 🚀:

Aqui está um exemplo completo de raspagem de dados climáticos de um site:

import requests
from bs4 import BeautifulSoup

url = 'https://www.example.com/climate'
response = requests.get(url)
html_content = response.content

soup = BeautifulSoup(html_content, 'html.parser')

data = soup.find_all('div', class_='data')
for d in data:
    date = d.find('span', class_='date').text.strip()
    precipitation = d.find('span', class_='precipitation').text.strip()
    print(date, precipitation)
